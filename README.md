## pixel-city
Get some photos 1 mi nearby from flickr
```
Double tap to choose area you want to see 40 photos nearby
Supported 3D touch
```
## API Used
Flickr.com
## Architecture

MVC (Model, View, Controller)

## Library Used

- Alamofire
- AlamofireImage

## Running

- Clone this repository
- Run `pod install`
- Open the xcworkspace
